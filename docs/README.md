# Documentation Directory

This folder contains various guides, policies, and reference materials for the PGVectorRAGIndexer project.

## `docs/internal/` Structure Diagram

> **⚠️ IMPORTANT CONTRIBUTOR NOTE regarding `docs/internal/`**
> 
> The `docs/internal/` directory is intentionally maintained as a checkout (via a git working tree or submodule) of a separate private repository containing commercial strategies and operational notes. 
> 
> **Editing Rules:**
> - The separate private repository is the canonical source for internal commercial and strategic documentation.
> - **Do not** commit changes to `docs/internal/` directly into this public repository's history unless explicitly syncing upstream.
> - Be aware that the contents of `docs/internal/` may be ahead or behind the public repository depending on when the sub-tree was last deliberately synced.
